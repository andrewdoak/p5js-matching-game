Match the Number

Are you fast enough to beat the computer (i.e. That thing projecting all the numbers at you)?

How to play >

Play on a computer (not mobile-ready...yet). Type the number you see when it comes at you.

If you miss, the screen will flash red. If the key you pressed matches the flying number, the screen will flash another (random) color.

A game = ten attempts.

Six matches or more = A WIN.

Three misses = A LOSS!

But you can try again when the game's over. PRO TIP > you dont have to try every round. Shhhh!
